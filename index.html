Nebula - An Issue Tracking Website for the Council of Stellar Management

Created in collaboration with active CSM and past CSM members, Nebula is a system that aims to provide voting, discussion and issue raising/collaboration functionality to the EVE Online community in a proper, dedicated web application to replace the current system of manually maintained wiki pages and forum threads.

Dependencies

Nebula requires Ruby 1.9.1 (1.8.7 should work, but is not tested) as well as the latest Rubygems. Nebula has quite a few dependencies in terms of gems, but these are all handled by bundler - just install the bundler gem, then run 'bundler install' in the application root. Nebula is developed with PostgreSQL in mind, but should work on MySQL and at a pinch SQLite.

Install

* Install Ruby 1.9.1 (from source is probably easiest) * Update rubygems with gem update --system * gem install bundler * gem install mongrel (Optional, but recommended) * Checkout code * Create database and configure connection settings in config/database.yml * bundle install * rake db:migrate * rake db:seed * ruby script/server Point browser at localhost:3000, you should have Nebula.
